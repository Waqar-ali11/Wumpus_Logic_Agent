# Wumpus_Logic_Agent
Wumpus World Logic Agent (AI Project)
Overview

This project implements a Knowledge-Based Agent for the Wumpus World environment using Propositional Logic and Inference Mechanisms.

The agent navigates a dynamic grid, perceives environmental cues (Breeze, Stench), and uses logical reasoning to deduce safe cells.

Features
Dynamic grid size (user-defined)
Random placement of Pits and Wumpus
Real-time percept generation
Knowledge Base using propositional logic
CNF conversion using SymPy
Safety inference using logical satisfiability
Interactive Streamlit UI




AI Concepts Used
Knowledge-Based Agents
Propositional Logic
CNF Conversion
Logical Inference
Resolution-based reasoning (conceptual)
Tech Stack
Python
Streamlit
NumPy
SymPy
Run Locally
pip install -r requirements.txt
streamlit run wumpus.py
